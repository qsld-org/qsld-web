# QSLD-Web

> [!WARNING]
> This program is in very early development. Most functionality is not implemented. Do not use.

This is the online playground for the QSLD quantum simulation library. If you are unfamiliar with qsld, you can check it out [here](https://github.com/qsld-org/qsld). Instructions for building, installing and running, will be posted once the program is in a more stable and functional state.
